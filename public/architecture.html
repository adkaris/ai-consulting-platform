Key Data Flows
1
Page request → Server Component fetches via Server Actions → Prisma queries SQLite → HTML streamed to browser
2
Assessment submit → calculateWeightedScore() → generateRecommendations() → saveAssessment() persists 8 domain scores
3
File upload → POST /api/upload → written to /uploads/ → Document record saved in DB → served via /api/files/[…path]
4
Analytics dashboard → getAllAssessments() aggregates domain scores → passed to AnalyticsClient → Recharts renders Radar / Bar / Pie
5
Gap analysis report → customer + assessment fetched → scores mapped to gap matrix → rendered as print-ready HTML / PDF
